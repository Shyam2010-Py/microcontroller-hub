Sensors

Three essential sensors every ESP32 beginner should know. Each card covers theory, wiring, code, and real-world uses.

🌡️

DHT11 — Temperature & Humidity

A low-cost digital sensor that measures ambient temperature and relative humidity using a thermistor and a capacitive humidity sensor.

Digital 3.3 V — 5 V Single-wire

Working Principle

The DHT11 uses a capacitive humidity sensor and a thermistor to measure the surrounding air. It outputs a calibrated digital signal on the data pin using a proprietary single-wire protocol.

Pin Connections

DHT11 VCCESP32 3.3 V
DHT11 DATAESP32 GPIO 4
DHT11 GNDESP32 GND
DATA — 3V310 kΩ pull-up

ESP32 Wiring

VCC → 3.3 V, GND → GND, DATA → GPIO 4 (with a 10 kΩ pull-up resistor to VCC).

Example Code

dht11.ino · uses Adafruit DHT sensor library 
#include "DHT.h"

#define DHT_PIN  4
#define DHT_TYPE DHT11

DHT dht(DHT_PIN, DHT_TYPE);

void setup() {
  Serial.begin(115200);
  dht.begin();
}

void loop() {
  float t = dht.readTemperature();   // °C
  float h = dht.readHumidity();      // %
  if (isnan(t) || isnan(h)) {
    Serial.println("Sensor read failed");
    return;
  }
  Serial.print("Temp: "); Serial.print(t); Serial.print(" °C  ");
  Serial.print("Humidity: "); Serial.print(h); Serial.println(" %");
  delay(2000);
}

Applications

  • Weather stations
  • Greenhouse monitoring
  • Smart HVAC control
  • Indoor air-quality dashboards

HC-SR04 — Ultrasonic Distance

Measures distance from 2 cm to 400 cm using ultrasonic sound waves. A staple for obstacle detection and ranging projects.

Digital 5 V logic 2 — 400 cm

Working Principle

The sensor emits a 40 kHz ultrasonic pulse from the trigger pin and waits for its echo. Distance is computed from the round-trip time:

distance (cm) = echo_time (µs) / 58

Sound travels ~0.034 cm/µs, so the round trip covers 2 × distance.

Pin Connections

VCC5 V
TRIGESP32 GPIO 5
ECHOESP32 GPIO 18 (via divider)
GNDGND

ESP32 Wiring

HC-SR04 outputs 5 V on ECHO. ESP32 GPIOs are 3.3 V tolerant, so use a voltage divider: ECHO → 1 kΩ → GPIO 18 → 2 kΩ → GND.

Example Code

hcsr04.ino 
#define TRIG_PIN 5
#define ECHO_PIN 18

void setup() {
  Serial.begin(115200);
  pinMode(TRIG_PIN, OUTPUT);
  pinMode(ECHO_PIN, INPUT);
}

long readDistanceCm() {
  digitalWrite(TRIG_PIN, LOW);
  delayMicroseconds(2);
  digitalWrite(TRIG_PIN, HIGH);
  delayMicroseconds(10);
  digitalWrite(TRIG_PIN, LOW);
  long us = pulseIn(ECHO_PIN, HIGH, 30000);
  return us / 58;     // cm
}

void loop() {
  Serial.print("Distance: ");
  Serial.print(readDistanceCm());
  Serial.println(" cm");
  delay(250);
}

Applications

  • Robotics obstacle avoidance
  • Parking sensors
  • Liquid level measurement
  • Smart doorbells and proximity alerts

IR Obstacle Sensor

A simple infrared reflection sensor that detects objects within a few centimeters. Perfect for line-followers and counters.

Digital out 3.3 V — 5 V 2 — 30 cm

Working Principle

An IR LED emits infrared light; a photodiode detects the light reflected off nearby objects. The on-board comparator outputs LOW when an object is detected and HIGH otherwise (active-low).

Pin Connections

VCCESP32 3.3 V
GNDESP32 GND
OUTESP32 GPIO 27

ESP32 Wiring

Three-wire hookup. Power the module from 3.3 V and read the OUT pin. Most boards have an on-board potentiometer to tune detection range.

Example Code

ir_obstacle.ino 
#define IR_PIN 27

void setup() {
  Serial.begin(115200);
  pinMode(IR_PIN, INPUT);
}

void loop() {
  int state = digitalRead(IR_PIN);  // LOW = object detected
  if (state == LOW) {
    Serial.println("Object detected");
  } else {
    Serial.println("Clear");
  }
  delay(150);
}

Applications

  • Line-follower robots
  • Object counters / tachometers
  • Security and trip-wire alarms
  • Automatic hand-sanitizer dispensers
