Blink LED

The "Hello World" of embedded systems. Blink the on-board LED with a fixed interval.

Beginner 5 min No external parts

Difficulty

Easy — the first project for every beginner. No wiring, no libraries, no soldering.

Components Needed

  • ESP32 DevKit board (1×)
  • USB cable (1×)

Wiring

None. The blue LED is soldered to GPIO 2 on the board.

Code

blink.ino
#define LED 2

void setup() {
  pinMode(LED, OUTPUT);
}

void loop() {
  digitalWrite(LED, HIGH);
  delay(1000);
  digitalWrite(LED, LOW);
  delay(1000);
}

Output

The on-board LED turns ON for 1 second, then OFF for 1 second, repeating forever.

Applications

  • Verifying the board works
  • Learning digital output & timing
  • Heartbeat / status indicators

Temperature Monitor

Read live temperature and humidity from a DHT11 and print it to the Serial Monitor every 2 seconds.

Intermediate 15 min DHT11

Difficulty

Intermediate — introduces a sensor library and reading real-world data.

Components Needed

  • ESP32 DevKit (1×)
  • DHT11 module (1×)
  • 10 kΩ resistor (1×)
  • Breadboard & jumper wires

Wiring

DHT11 VCCESP32 3.3 V
DHT11 DATAESP32 GPIO 4
DHT11 GNDESP32 GND
DATA — 3V310 kΩ pull-up

Code

temp_monitor.ino · Adafruit DHT lib required
#include "DHT.h"
#define DHT_PIN  4
#define DHT_TYPE DHT11
DHT dht(DHT_PIN, DHT_TYPE);

void setup() {
  Serial.begin(115200);
  dht.begin();
}

void loop() {
  float t = dht.readTemperature();
  float h = dht.readHumidity();
  if (isnan(t) || isnan(h)) {
    Serial.println("Read failed");
    return;
  }
  Serial.printf("Temp: %.1f °C   Humidity: %.1f %%\n", t, h);
  delay(2000);
}

Output

Temp: 26.0 °C   Humidity: 54.0 %
Temp: 26.1 °C   Humidity: 53.8 %
Temp: 26.1 °C   Humidity: 53.7 %

Applications

  • DIY weather station
  • Server room monitoring
  • Greenhouse automation
  • Smart-fan controller trigger

Distance Meter

Use an HC-SR04 ultrasonic sensor to measure distance in centimeters and warn when something comes too close.

Intermediate 20 min HC-SR04

Difficulty

Intermediate — adds pulseIn(), voltage-divider wiring, and threshold logic.

Components Needed

  • ESP32 DevKit (1×)
  • HC-SR04 sensor (1×)
  • 1 kΩ & 2 kΩ resistors (voltage divider)
  • Breadboard & jumper wires
  • Optional: LED + 220 Ω for visual alert

Wiring

HC-SR04 VCCESP32 5 V (VIN)
HC-SR04 GNDESP32 GND
HC-SR04 TRIGESP32 GPIO 5
HC-SR04 ECHO1 kΩ → GPIO 18 → 2 kΩ → GND
LED (+)GPIO 2 (built-in)

Code

distance_meter.ino
#define TRIG 5
#define ECHO 18
#define LED  2
#define THRESHOLD 20   // cm

long readCm() {
  digitalWrite(TRIG, LOW);  delayMicroseconds(2);
  digitalWrite(TRIG, HIGH); delayMicroseconds(10);
  digitalWrite(TRIG, LOW);
  return pulseIn(ECHO, HIGH, 30000) / 58;
}

void setup() {
  Serial.begin(115200);
  pinMode(TRIG, OUTPUT);
  pinMode(ECHO, INPUT);
  pinMode(LED, OUTPUT);
}

void loop() {
  long d = readCm();
  Serial.print("Distance: "); Serial.print(d); Serial.println(" cm");
  digitalWrite(LED, (d > 0 && d < THRESHOLD) ? HIGH : LOW);
  delay(200);
}

Output

Distance: 84 cm
Distance: 65 cm
Distance: 18 cm  ← LED ON (warning)
Distance: 12 cm  ← LED ON (warning)

Applications

  • Reverse parking sensor
  • Smart doorbell that triggers on approach
  • Liquid level in a tank
  • Obstacle-avoiding robot
