What is ESP32?

The ESP32 is a low-cost, low-power system-on-chip (SoC) microcontroller developed by Espressif Systems. It comes with built-in Wi-Fi and Bluetooth, making it ideal for Internet of Things (IoT) projects, embedded systems, and rapid prototyping.

It is the successor to the popular ESP8266 and adds more power, more GPIOs, Bluetooth, and hardware encryption.

At a glance ESP32 = Arduino-level simplicity + Wi-Fi + Bluetooth + dual-core processing.

Features

  • Dual-core Xtensa LX6 processor, up to 240 MHz
  • Wi-Fi 802.11 b/g/n (2.4 GHz)
  • Bluetooth 4.2 (Classic + BLE)
  • 520 KB SRAM, supports external flash/PSRAM
  • 34 programmable GPIO pins
  • 12-bit ADC, 8-channel
  • Touch sensors, Hall sensor, temperature sensor
  • SPI, I2C, UART, I2S, CAN
  • PWM on almost every GPIO
  • Operating voltage: 3.3 V

Applications

  • IoT devices and smart home automation
  • Wireless sensor networks
  • Wearable electronics
  • Robotics and motor control
  • Data logging and environmental monitoring
  • BLE beacons, smart locks, asset trackers
  • Wi-Fi controlled relays, lights, and appliances

Your First Program

Before uploading, install the ESP32 board package in the Arduino IDE:

  1. Open Arduino IDE → File → Preferences
  2. Paste this URL into Additional Board URLs:
    https://raw.githubusercontent.com/espressif/arduino-esp32/gh-pages/package_esp32_index.json
  3. Go to Tools → Board → Board Manager → search esp32 → install
  4. Select ESP32 Dev Module as your board
  5. Pick the correct Port, then click Upload

Blink LED

The classic first program. The on-board LED on GPIO 2 blinks once per second.

blink.ino 
// ESP32 Blink — on-board LED on GPIO 2
#define LED_PIN 2

void setup() {
  pinMode(LED_PIN, OUTPUT);
}

void loop() {
  digitalWrite(LED_PIN, HIGH); // ON
  delay(1000);
  digitalWrite(LED_PIN, LOW);  // OFF
  delay(1000);
}

What you learned

  • pinMode() sets a pin as input or output
  • digitalWrite() drives the pin HIGH or LOW
  • delay() pauses the program in milliseconds
  • setup() runs once, loop() runs forever
